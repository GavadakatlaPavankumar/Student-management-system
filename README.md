# 🎓 Student Management System

A menu-driven console-based Student Management System developed using **Python** and **CSV File Handling**. The application allows users to efficiently manage student records through CRUD (Create, Read, Update, Delete) operations while storing data persistently in a CSV file.

---

## 📌 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by Student ID
- ✏️ Update Student Details
- 🗑️ Delete Student Record
- 📊 Count Total Students
- 💾 Persistent data storage using CSV files
- ✅ Duplicate Student ID validation

---

## 🛠️ Technologies Used

- Python 3
- CSV File Handling
- File I/O Operations
